> `$HOME/.bashrc`

 * Define `touchp` alias
 * Define the `gac` alias
 * Display git branch in terminal
 * Colorize Terminal with ANSI

> `$HOME/.ssh/config`

 * Define shortcuts for common `ssh` commands, by host and key-pair.
 * Add Sample to expose a remote port on localhost port.
